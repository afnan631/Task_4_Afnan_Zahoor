📋 Project 4: Form Design & Validation
 
📌 About
A fully interactive Registration Form built with React.js. It validates every field in real-time using custom JavaScript logic following the Input-Process-Output (IPO) model — user types (Input), validation runs (Process), error or success message appears (Output).

✨ Features
Real-time validation with error and success messages
Password strength meter (Weak → Strong)
Show/Hide password toggle
Confirm password cross-field check
Terms & Conditions checkbox validation
Success screen on clean submission
Dark glassmorphism UI

🚀 How to Run
# 1. Install the project folder from github in zip and then extract the project folder.open that folder with VS Code Editor.

# 2. Install packages and run
npm install
 
# 3 and run
npm run dev
